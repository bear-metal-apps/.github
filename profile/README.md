# Bear Metal Apps

Welcome to the **Bear Metal Apps** GitHub organization!
This org is home to the apps and tools we build for our team, including:

- Our scouting system, Beariscope and Pawfinder
- Servers and shared libs
- Other utilites for the team

## About Us

Bear Metal is a high school robotics team based in Maple Valley, WA, competing in the First Robotics Competition. Our subteam, Apps, focuses on creating software to support strategy, scouting, and team operations.
